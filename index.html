GAME DEALS
SCRAPER

Scrapes live game discounts from Steam & cross-store retailers, tracks price history, and alerts you when a wishlist game hits your target price.

View source on GitHub → Try the live demo ↓
01

Scrapes real listings

BeautifulSoup pulls Steam's Specials page by genre tag; the CheapShark API fills in GOG, Epic, Humble, and more.

02

Tracks price history

Every price change is logged to SQLite, powering a per-game sparkline so you can see if a "deal" is actually a low.

03

Watches for you

Set a target price on any title. The scraper checks the watchlist on every run and flags what's hit its target.

Run the full app locally

git clone https://github.com/Jvn1604/game-deals-scraper.git
cd game-deals-scraper
pip install -r requirements.txt
python db.py
python scraper.py   # optional — refresh live data
python app.py        # open http://127.0.0.1:5000
